# Simple ERC-20 Token

A high-quality, standard-compliant implementation of an ERC-20 token on the Ethereum blockchain. This repository is designed for developers looking to launch a basic utility or governance token with best-practice security standards.

## Features
* **Standard Compliant:** Fully compatible with all ERC-20 wallets and exchanges.
* **OpenZeppelin Integrated:** Leverages industry-standard libraries for security.
* **Fixed Supply:** Mints the total supply to the creator upon deployment.

## Getting Started
1. Install dependencies using `npm install`.
2. Update the token name and symbol in `SimpleToken.sol`.
3. Deploy to your network of choice using the provided `deploy_token.js` script.
